# bnwunshit
# Browser plugin to remove shit from bnw pages in runtime

INSTALLATION:
1) git clone https://github.com/afwbkbc/bnwunshit.git
Firefox:
2) type about:debugging in address bar
3) check "Enable add-on debugging" checkbox
4) click "Load Temporary Add-on"
5) navigate to cloned directory, then "src" subdirectory, then select any file ( i.e. manifest.json )
Chrome:
2) go to Settings -> Extensions
3) enabled "Developer mode" slider at top right corner
4) click "Load unpacked" button below top left corner
5) navigate to cloned directory, then "src" subdirectory, then click "Select folder"

USAGE:
Plugin will detect *//bmw.im/* urls and remove shit automatically.
